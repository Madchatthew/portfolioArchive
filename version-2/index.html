About Me

I am a nurse of six and a half years. The last three years I have been a Travel Registered Nurse. My family travels with me. We have had the opportunity to experience parts of Alaska to the eastern seaboard. Throughout my career, I have been working on and helping others with websites. I started to learn more web development languages and have found that I have a deep passion and joy for web development. I enjoy creating new sites as well as troubleshooting and improving existing sites/technologies. I look forward to making my new found passion into a lifelong career.

Public API Search
This project is a fullstack application using nodejs. You can search for existing public APIs. You can save to a favorites list. This app is a work in progress and updates will be continued over time. See it live at https://publicapi.chadjessen.com
Theme Light Shadow
Theme extension created for a project called Flarum.io, which is a modern community forums. Flarum requires Composer to install. Node and NPM is required to created extensions. Github is used for version control.
whatsonyourscrubs.com
Whatsonyourscrubs.com was created using Flarum. It is located on a Digitalocean droplet, which is a virtual server that runs Ubuntu Server 18.04. Updates to the server and site are completed remotely. Currently this server is running Nginx, PHP7.4 and Mariadb.
Grandforkschurch.com
Grandforkschurch.com was setup using wordpress. The theme in use was customized for appearance and functionality. Grandforkschurch.com is hosted on Siteground. Control of the site was handed off to a church member in 2017.
Nordvickarms.com
Nordvickarms includes woocommerce for the sales of products. The owner is able to use blogposts for "Latest News" section. Currently the inventory database is updated via a CSV file. API imports will be added soon. I currently update and maintain this site. Nordvickarms is hosted on Siteground.
Hollyjessen.com
Hollyjessen.com was setup using wordpress and a theme which was customized for appearance and functionality. I currently update and maintain this site. This site is hosted on a Digitalocean Droplet.
